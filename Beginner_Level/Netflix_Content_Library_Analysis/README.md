# Netflix Content Library Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix content library dataset using Python. The goal is to uncover insights regarding content types, regional distributions, genre popularity, and other key trends within Netflix's vast catalog of movies and TV shows.

Understanding the composition of this content library can help identify content trends, audience preferences, and regional distribution patterns. This analysis provides actionable business insights that could support content acquisition strategies, audience engagement, and regional expansion decisions.

## Analysis Objectives

The analysis aims to answer several key questions and explore various aspects of the dataset:
- **Movies vs. TV Shows:** Analyze the distribution of different content types.
- **Regional Analysis:** Identify top content-producing countries on the platform.
- **Genre Exploration:** Discover the most common genres available.
- **Temporal Trends:** Study the patterns of content additions over time.
- **Duration Analysis:** Examine movie duration and TV show season patterns.
- **Audience Targeting:** Understand audience targeting through content ratings.

## Libraries Used

- `pandas` for data manipulation
- `numpy` for numerical computing
- `matplotlib` and `seaborn` for data visualization

## Workflow

1. **Load Data**: The dataset `netflix_titles.csv` is loaded and inspected.
2. **Exploratory Data Analysis (EDA)**: Understanding dataset dimensions, dealing with missing values, and exploring data distributions.
3. **Trend Analysis**: Visualizing content types, regional distributions, and genre popularity.
4. **Temporal Analysis**: Analyzing the addition of new content over time.
5. **Audience & Duration Analysis**: Investigating audience target demographics based on ratings and content duration.

## How to Run

Open the `Netflix_Content_Library_Analysis.ipynb` Jupyter Notebook and run the cells sequentially. Make sure you have the required libraries installed and the dataset `netflix_titles.csv` in the appropriate directory as referenced in the notebook.
